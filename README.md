# CarApp 🚗

Gestionnaire de flotte automobile simple et efficace.

## 🌟 Fonctionnalités
- **Home** : Tableau de bord de navigation.
- **Listing** : Vue complète de l'inventaire.
- **CRUD** : Ajout et suppression de véhicules.

## 🛠 Tech Stack
- **Backend** : Java 17, Spring Boot 3, JPA/Hibernate.
- **Frontend** : HTML5, Thymeleaf, **Pur CSS** (aucun framework).
- **Base de données** : H2 (in-memory).

## 🚀 Lancement Rapide
1. Lancer l'application via votre IDE.
2. Accès : [http://localhost:8080/](http://localhost:8080/)
3. Console DB : [http://localhost:8080/h2-console](http://localhost:8080/h2-console) (JDBC: `jdbc:h2:mem:testdb`)
